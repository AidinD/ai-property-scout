Vad extensionen gör

AI Property Scout är en Chrome-extension som analyserar fastighetsannonser på Hemnet.se och Booli.se. Extensionen läser av annonsdata på sidan du besöker och skickar delar av den till AI-tjänster för analys.

Data som behandlas lokalt

Följande sparas enbart i din webbläsare via chrome.storage.local och lämnar aldrig din enhet:

  • Fastighetsdata från annonser du besöker (adress, pris, fakta)
  • AI-analyser och PDF-analyser du genomför
  • Din portfolio med sparade objekt
  • Din API-nyckel (om du använder Consumer-läget)
  • Inställningar (vald AI-leverantör, räntesats m.m.)

Data som skickas till tredje part

Anthropic / OpenAI — när du analyserar en annons eller ett dokument skickas mäklartext och fastighetsfakta till den AI-leverantör du valt. Ingen personuppgift ingår. Se respektive leverantörs policy: Anthropic, OpenAI.

Cloudflare Workers — broker-tiers skickar AI-anrop via en Cloudflare Worker som proxy. Anropen innehåller enbart annonstext och fastighetsfakta.

Supabase — extensionen skickar anonyma användningshändelser (t.ex. "analys startad", "PDF analyserad") för att förstå hur produkten används. Ingen personidentifierbar information ingår. Ett slumpmässigt anonymt ID genereras och lagras lokalt.

Data vi inte samlar in

  • Namn, e-postadress eller annan personuppgift
  • Browsing-historik utanför Hemnet och Booli
  • Betalningsinformation (hanteras av Lemon Squeezy)
  • Din IP-adress eller exakta plats

Behörigheter

Extensionen begär följande Chrome-behörigheter:

  • hemnet.se, booli.se — för att läsa annonsdata och injicera sidebaren
  • storage — för att spara din portfolio och inställningar lokalt
  • optional host permissions — begärs per domän när du hämtar information från mäklarsidor, enbart om du aktivt klickar på "Hämta mer info"
